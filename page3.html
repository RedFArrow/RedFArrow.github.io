MultiMC Installation

How to install and use MultiMC

Downloading MultiMC

Firstly, you need to Download multiMC. The windows build of which is here. 
This link will download a file named mmc-stablewin32.zip
Open this, and inside will be a folder named MultiMC. Right click this folder, and select copy.

Now, in file explorer, find your documents folder. [You can do this by typing in Documents in the address bar of the file explorer]. Right click anywhere and paste the MultiMC folder in here.

Open the MultiMC folder, and find the MultiMC.exe file. Right click on this file and create a shortcut. This will create a near identical "MultiMC.exe - Shortcut" file, you can put this whereever you'd like to open multimc from (such as your desktop). 

Using MultiMC

After you have gone through the first time setup MultiMC will open into a blank window with several buttons.

MultiMC has a lot of functionality but i'll run through two basic use cases.

Creating a vanilla instance
You can do this by pressing Add Instance, selecting the version, naming the version at the top, then pressing OK. 

You can install forge or fabric to this instance by right clicking on it and selecting Edit Instance, and then on the right hand side of the version tab there is a 'Install forge' and 'Install fabric' button. 


Installing a pre-zipped mod pack
You can install a MultiMC compatible mod pack file by simply dragging the zip file into the window.
Alternatively you can press Add instance, Import from zip and then pasting the URL of the file.
